Changes

Gets or sets the URL the web site's change tracking endpoint.

Declaration
public Uri Changes { get; set; }
Property Value
Type Description
Uri

A that represents the URL the web site's change tracking endpoint.
Remarks

When a feed that contains this element updates, it pings a server that updates this file. The presence of this element indicates to aggregators that they only have to read the changes file to see if this feed has updated. If several feeds point to the same changes file, aggregators have to do less polling, resulting in better use of server bandwidth and faster scans. See http://www.xmlrpc.com/weblogsComForRss for technical details.
