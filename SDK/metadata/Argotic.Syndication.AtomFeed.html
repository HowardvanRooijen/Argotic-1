Class AtomFeed

Represents an Atom syndication feed, including metadata about the feed, and some or all of the entries associated with it.

Syntax
public class AtomFeed : object, ISyndicationResource, IAtomCommonObjectAttributes, IExtensibleSyndicationObject
Remarks

Atom is an an XML-based Web content and metadata syndication format that describes lists of related information known as feeds. Feeds are composed of a number of items, known as entries, each with an extensible set of attached metadata.

This implementation conforms to the Atom 1.0 specification, which can be found at http://www.atomenabled.org/developers/syndication/atom-format-spec.php.

If multiple AtomEntry objects with the same Id value appear in an Atom Feed Document, they represent the same entry. Their UpdatedOn timestamps should be different. If an Atom Feed Document contains multiple entries with the same Id, Atom Processors may choose to display all of them or some subset of them. One typical behavior would be to display only the entry with the latest UpdatedOn timestamp.

When an Atom Document is relocated, migrated, syndicated, republished, exported, or imported, the content of its universally unique identifier must not change.
