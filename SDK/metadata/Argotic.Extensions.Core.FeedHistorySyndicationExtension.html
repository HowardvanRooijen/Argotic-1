public class FeedHistorySyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The FeedHistorySyndicationExtension extends syndicated content to specify three types of syndicated Web feeds that enable publication of entries across one or more feed documents. This includes paged feeds for piecemeal access, archived feeds that allow reconstruction of the feed's contents, and feeds that are explicitly complete. This syndication extension conforms to the Feed Paging and Archiving 1.0 specification, which can be found at http://www.ietf.org/rfc/rfc5005.txt.

Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
