Remarks

The SyndicationExtension abstract class is provided to reduce the difficulty of implementing custom syndication extensions. While implementers are free to implement their custom syndication extensions by implementing the ISyndicationExtension interface, it is recommended that custom syndication extensions inherit from the SyndicationExtension base class.

If you choose to not inherit from the SyndicationExtension abstract base class, please be aware that the SyndicationExtensionAdapter class internally calls the method, and so any custom syndication extension will need to have the appropriate and attributes applied as necessary.

This method will return a that has a namespace added to it using the XmlPrefix and XmlNamespace of the extension unless the supplied already has an XML namespace associated to the XmlPrefix, in which case the associated XML namespace is used instead. Type Description
XmlSchema

A object that represents an in-memory representation of an XML Schema as specified in the World Wide Web Consortium (W3C) XML Schema Structures and Datatypes specifications. The default return value for this method is a null reference (Nothing in Visual Basic).
Remarks

When serializing or deserializing an object, the class does not perform XML validation. For this reason, it is often safe to omit schema information by providing a trivial implementation of this method, for example by returning a null reference (Nothing in Visual Basic).

Some .NET Framework types as well as legacy custom types implementing the interface may be using instead of . In this case, the method returns an accurate XML schema that describes the XML representation of the object generated by the WriteXml(XmlWriter) method.
