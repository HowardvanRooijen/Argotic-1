Remarks

This constructor is meant to be used when creating an explicit author. Explicit data is for items that are explicitly added by a user to represent something. For example, a user could edit their own APML file and add items they know they're interested in. For this reason the From and UpdatedOn properties are not necessary for explicit data items, because it's a manual process.

Parameters
Type Name Description
System.String key

The unique key for this author.
System.Decimal value

The decimal score of this author.
System.String from

The name of the entity that contributed this author.
DateTime utcUpdatedOn

A object that indicates the last time this author was updated.
Remarks

This constructor is meant to be used when creating an implicit author. Implicit data is added by machines/computers that try to make some informed guesses about the things that you are interested in. This stuff will change over time and are added with a certain degree of confidence that may have a decay in certain applications. For this reason it is important to keep a track of when things were added/modified.
